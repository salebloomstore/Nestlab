## 📌 Overview

Enterprise-ready fullstack system built with:

- 🚀 NestJS (Backend API)
- 🟢 Node.js runtime
- 🍃 MongoDB database
- 🧩 Mongo Express (DB Admin)
- 🌐 Nginx (Reverse Proxy)
- 🐳 Docker / Docker Compose
- 🎨 jQuery / Bootstrap / AJAX frontend support
